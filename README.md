## 🛠 Troubleshooting

If you encounter issues, review these common solutions:

*   **Game refuses to start:** Ensure the path to the game folder is correct. If the tool fails to patch the files, try running the client as an administrator. Right-click the desktop icon and select "Run as administrator."
*   **Upscaling options are greyed out:** Check if your graphics card supports the selected technology. Not all GPUs support every upscaling type. Re-scan your game library to confirm the tool sees your hardware correctly.
*   **Game performance stays the same:** Open the game settings menu while inside the game. Ensure you have activated the upscaling option in the game's own graphics settings, as the patch simply unlocks the capability for the game to see it.
*   **Anti-virus alerts:** Some security programs might flag the patching process. These tools modify game binaries, which triggers suspicious behavior warnings. Add an exclusion folder for your game library in your anti-virus settings to stop these alerts.
