## Overview
OptiScaler Client FSR4 is a dedicated client for AMD FidelityFX Super Resolution 4 technology. It brings advanced upscaling and frame generation capabilities with excellent image quality and minimal artifacts. The tool is optimized for modern games, offering flexible configuration, sharp visuals, and performance gains. Designed for gamers who want to push higher resolutions and smoother frame rates while maintaining visual fidelity on Windows 11 and Linux.
